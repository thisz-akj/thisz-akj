## About Me

I am a perception engineer with hands-on experience building and deploying real-time multi-model vision systems on edge hardware for autonomous robotic platforms.

My work spans the full perception pipeline — from model development to production deployment — including detection, segmentation, tracking, pose estimation, and sensor fusion under real-world constraints.
